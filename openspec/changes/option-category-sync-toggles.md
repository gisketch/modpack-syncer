# Change: option-category-sync-toggles

## Summary

Add per-category option sync toggles in Options Review.

## Requirements

1. Options Review MUST show a sync switch for each options category near `SHOW IGNORED`.
2. Category sync switches MUST default on each time sync review opens.
3. If a category sync checkbox is off, sync MUST leave local keys in that category unchanged.
4. Ignored key behavior MUST still override enabled categories.
