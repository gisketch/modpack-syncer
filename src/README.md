# Source

React frontend source.

## Shape

- `routes/`: thin route entry points.
- `features/`: feature-owned UI and helpers.
- `components/ui/`: reusable UI primitives.
- `stores/`: shared Zustand stores.
- `hooks/`: shared hooks.
- `lib/`: shared frontend utilities and typed Tauri calls.
- `styles/`: global CSS.

Keep behavior changes aligned with `openspec/specs/`.
